# 14장: 다중상수 SFE 실험 설계와 검증 프로토콜(신설)

## 1. 원칙

- 무튜닝, 섹터 단일책임, 순환 논리 배제, 반증 우선.

## 2. 미시(ε_mass=0.37 고정) — 3건 신속 검증

1) 거대분자 간섭(C60↑)
   - 예측: 결맞음 시간 ×1.587
   - 스윕: 압력·온도·기체종, 배율 불변이면 PASS

2) LIGO 열잡음
   - 예측: 열잡음 PSD ×1.587
   - 주파수 대역: 10–40 Hz 우선, 런(O3→O4) 간 배율 일관 확인

3) 우주선 뮤온
   - 예측: 관측 수명 ×1.259(에너지 높을수록 선명)
   - 고도·에너지 스윕, 기상 교정 포함

## 3. 거시(ε_grav 1점 고정) — 전구간 교차검증

1) S8 또는 S8_lens 한 점으로 ε_grav 결정
2) fσ8(z), S8_lens 전 구간 검증(≤1σ 목표)
3) 초기우주: μ(a≪1)→1 확인(BBN/CMB)
4) 가변상수: g_C≈0 상한 선언(원자시계·쿼이저 링크)

## 4. 장치별 체크리스트

- 옵토메카닉스(SQL): 교차점 주파수 유지, SQL 높이만 ×1.259
- 원자간섭계: 라만/브래그 반동 주파수 ×1.587, 동시성 검사
- 큐비트 T2 서베이: 세대·공정 간 평균 레벨 이동(상수배)

## 5. 리포트 포맷(학계 제출용)

- 입력 고정 명시(ε_mass=0.37, ε_grav 한 점)
- 예측 표(배율·오차), 관측 표(값·오차), Δ/σ, PASS/FAIL
- 실패 시 다음 조치(데이터 결함 vs 이론 폐기) 분기 규칙

## 6. 윤리·안전

- 가변 중력/관성 ‘응용’ 주장은 실험적 검증 전 금지
- 데이터 사후 가공 금지(사전 고정 원칙)

## 7. 예측 타겟 & PASS 기준(무튜닝)

- 공통 고정: $\epsilon_{\text{mass}}=2\Omega_\Lambda-1=0.37$, 배율 $1/(1-\epsilon)=1.5873$, $1/\sqrt{1-\epsilon}=1.2590$
- 미시
  - 결맞음(C60↑): 1.587±0.05 (압력·온도·기체종 전 범위에서 배율 불변; 기울기 |s|<1%/스윕)
  - LIGO 열잡음(10–40 Hz): 1.587 (허용 1.45–1.70; |d lnR/df|<0.02/Hz)
  - 뮤온 관측 수명(E>20 GeV): 1.259 (허용 1.20–1.32; 에너지 기울기 ~0)
  - 원자간섭계 반동 주파수(Rb/Cs): 1.587±0.05
  - 옵토메카닉스 SQL 높이: 1.259 (허용 1.23–1.29; 교차점 주파수 변동 무)
- 거시(한 점 고정 → 전 구간)
  - 고정: $S8_{\rm lens}=0.776$로 $\epsilon_{\text{grav}}$ 결정
  - 교차 검증: $f\sigma_8(0.32)=0.438$, $f\sigma_8(0.57)=0.447$, $f\sigma_8(0.70)=0.442$ (각 |Δ|≤0.3σ)
  - KiDS $S8_{\rm lens}$: 0.761–0.783 범위 내
